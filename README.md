# Pizza Delivery Application

## About the Project
This project is a Pizza Delivery web application developed as part of a Web Development & Designing internship program.

The application focuses on creating a clean, responsive, and user-friendly interface that represents the workflow of an online pizza ordering system. It demonstrates frontend design principles and user interaction using basic web technologies.

## Features
- Responsive landing page design
- Pizza menu displayed using interactive cards
- Add to cart functionality with live cart count
- User login interface with frontend validation
- Smooth order flow and checkout confirmation (simulated)
- Clean navigation and structured layout
- Mobile-friendly design using CSS media queries

## Order Flow & User Experience
The application follows a smooth and intuitive order flow. Users can browse pizzas, add items to the cart, and proceed through a simulated checkout process.

Order confirmation and checkout interactions are handled seamlessly on the frontend using JavaScript, providing clear feedback to users at each step. This ensures a consistent and easy-to-understand experience while showcasing the typical workflow of a food delivery application.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Scope
This project focuses on frontend development and workflow simulation. Backend functionalities such as database integration, real authentication, payment gateways, and email notifications are not implemented and are represented using placeholder logic for demonstration purposes.
